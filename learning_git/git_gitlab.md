# Gitlab

***GitLab*** est un logiciel de **forge** (outil qui permet à plusieurs personnes de travailler sur un même projet, et qui contient généralement des systèmes de gestion de version, gestion de tâches, de wiki/cdocumentation, suivi de bugs, intégration continue,...), basé sur ``git``

+ GitLab permet de faire:
    - de l'intégration continue
    - du déploiement continue
    - la gestion de container directement dans gitlab (sans avoir à passer par le docker hub) 
    - de l'ajout de snippets
    - de la gestion de projet en équipe, avec les kanban (les todo list avec des colonnes à faire, en cours, terminé,... )
    - la gestion des issues
